Switch off and on at plug socket Can thermostat change temperature? Is BRU red light ON above radiator? Resident can't reset thermostat Check if heating comes ON manually Resident uses override button if urgent Call Switchee for hardware replacement

1. Switch off and on at plug socket

Ask resident to turn OFF Switchee at wall plug, wait a minute, and turn it back ON.

2. Can thermostat change temperature?

Test if the resident can turn up the thermostat temperature manually.

3. Is BRU red light ON above radiator?

Check if the BRU red light above the radiator symbol turns ON, confirming heating is being called for.

4. Resident can't reset thermostat

Guide them to call Switchee support immediately if they cannot control the screen.

5. Check if heating comes ON manually

Instruct to use override button on the BRU to turn heating/hot water manually ON if needed.

6. Resident uses override button if urgent

Advise keeping heating ON with override until Switchee can arrange a repair/replacement.

7. Call Switchee for hardware replacement

Resident should contact Switchee immediately if thermostat remains frozen/unresponsive after reset.
