🔥 Situation 5: Heating Coming On Without Asking

← Back to Main Menu
Has a timer been set? Can resident turn heating OFF at thermostat? Check red light on BRU above radiator Turn OFF using override or boiler switch Reset boiler if needed Organize repair visit if issues persist

1. Has a timer been set?

Ask if a timer is active. If yes, Switchee support can assist adjusting it.

2. Can resident turn heating OFF at thermostat?

If no, continue troubleshooting via manual override or isolation switch.

3. Check red light on BRU above radiator

Check if the BRU shows a heating ON light even when thermostat is OFF.

4. Turn OFF using override or boiler switch

Press override button on BRU or turn OFF boiler at isolator to manually stop heating.

5. Reset boiler if needed

After switching boiler back ON, confirm if heating stays OFF now.

6. Organize repair visit if issues persist

If heating still comes ON unexpectedly, book engineer visit and notify Switchee support.
